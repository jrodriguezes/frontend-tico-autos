# TicoAutos - Frontend

## Universidad Técnica Nacional
**Carrera:** Ingeniería del Software
**Curso:** Programación en Ambiente Web II
**Profesor:** Bladimir Arroyo

### Descripción
Este proyecto es la aplicación cliente para la plataforma TicoAutos, diseñada para la publicación y consulta de automóviles en venta.

### Requerimientos Implementados
- [x] Registro y Login de Usuarios
- [x] Búsqueda y Filtrado de Vehículos (Marca, Modelo, Año, Precio, Estado)
- [x] Visualización de Detalles (Integración parcial)
- [x] Sistema de compartición de enlaces
- [x] Interfaz responsiva con diseño premium
